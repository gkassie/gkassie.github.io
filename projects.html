MyDestinationExpress

MyDestinationExpress is a comprehensive web-based software solution designed for Destination Express LLC. It streamlines operations for shippers, couriers, drivers, and dispatchers by providing distinct functionalities for each user type. Shippers can register, request quotes, create and track orders, and request cancellations. Couriers can apply to work with Destination Express and, upon approval, access a specialized driver dashboard to view, take, and manage orders. The dispatcher dashboard allows dispatchers to manage quote and courier requests, cancellations, and dispatch drivers for orders.The driver dashboard allows drivers to view order information, change order status, and provide proof of delivery. The project was developed using Python, the Django framework, and front-end technologies like HTML, CSS, and JavaScript, with MySQL serving as the database.
